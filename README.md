# Projeto Cloud

Script para criar toda a infraestutura de uma aplicação na AWS.

## Bibliotecas necessárias

1) Boto3

```
pip3 install boto3
```

## Execução

1) No terminal execute

```
source set_env
```

2) Ainda no terminal, execute

```
python3 awsAPI.py
```
